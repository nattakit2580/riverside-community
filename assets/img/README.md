# รูปภาพที่ต้องเตรียม (Image Assets Guide)

วางไฟล์รูปในโฟลเดอร์นี้ แล้วตั้งชื่อตามด้านล่าง เว็บจะดึงไปแสดงอัตโนมัติ
ระหว่างที่ยังไม่มีรูปจริง เว็บจะใช้ภาพ placeholder จาก Unsplash ชั่วคราว

## รูปหลัก (จำเป็น)
| ชื่อไฟล์ | ใช้ที่ | ขนาดแนะนำ | เนื้อหา |
|----------|--------|-----------|---------|
| `hero.jpg`        | Hero เต็มจอ | 2400×1350 (16:9) | แม่น้ำแควน้อย + พระอาทิตย์ตก |
| `positioning.jpg` | section แนะนำ | 1600×1200 | บ้านไม้ริมน้ำ / สนามหญ้า |
| `house-1.jpg`     | บ้านหลัง 1 | 1200×900 (4:3) | บ้านพักหลังที่ 1 |
| `house-2.jpg`     | บ้านหลัง 2 | 1200×900 | บ้านพักหลังที่ 2 |
| `house-3.jpg`     | บ้านหลัง 3 | 1200×900 | บ้านพักหลังที่ 3 |
| `house-4.jpg`     | บ้านหลัง 4 | 1200×900 | บ้านพักหลังที่ 4 (ถ้ามี) |

## แกลเลอรี (Gallery)
| ชื่อไฟล์ | เนื้อหา |
|----------|---------|
| `gallery-1.jpg` | คนนั่งทำงานริมแม่น้ำ |
| `gallery-2.jpg` | คู่รักนั่งดื่มกาแฟ |
| `gallery-3.jpg` | สนามหญ้า / ธรรมชาติ |
| `gallery-4.jpg` | บ้านไม้ตอนเย็น |
| `gallery-5.jpg` | บรรยากาศแม่น้ำยามเช้า |
| `gallery-6.jpg` | มุมพักผ่อน |

## เคล็ดลับ
- ใช้ไฟล์ **.jpg** สำหรับภาพถ่าย (เล็กกว่า) คุณภาพ 80–85%
- บีบขนาดก่อนอัปโหลดที่ https://squoosh.app เพื่อให้เว็บโหลดเร็ว
- ภาพ Hero แนวนอนสว่าง ๆ จะดูโปร่งและ "Peaceful" ที่สุด
